Importa CSV
Importa contatti da un file CSV Brevo
Formato atteso

Il file CSV deve avere queste colonne (separate da virgola):
EMAIL, FIRSTNAME, LASTNAME, PHONE, TAG, INTERESTS

Il campo INTERESTS deve usare il formato Brevo: ['lingue'|'traduzioni']
Il campo PHONE verrà copiato in SMS, WHATSAPP e LANDLINE_NUMBER all'esportazione.

Come funziona il sync

Pull — scarica il file contacts.json dal repo e unisce i contatti con quelli locali (merge per email).
Push — carica i contatti locali sul repo. Se il file esiste già, fa prima un pull e poi sovrascrive con il merge.
In questo modo più persone possono lavorare in parallelo senza perdere dati.
